   ```bash

   ```

2. Open `mail.html` in your browser

## Customization

Edit these variables in `mail.css`:
```css
:root {
  --glass-blur: 12px;
  --glass-opacity: 0.1;
  --animation-duration: 0.4s;
}
```

## Deployment

Simply upload these files to your GitHub Pages:
- mail.html
- mail.css
